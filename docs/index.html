what is ravencli

An automated recon and hardening copilot for your shell.

RavenCLI turns common security tasks into guided flows: choose a target, pick a profile and let the tool orchestrate recon, scanning and config checks while you stay in control of every step.

Under the hood, RavenCLI builds on the OpenCode agent architecture, so you can plug in your own models and providers, extend modules and script repeatable audits for your infrastructure and lab environments.

Terminal‑native interface
Navigate scans, results and actions via a fast keyboard‑driven layout that feels at home in your shell.
Ethical hacking focus
Optimized for defensive security and authorized testing so you can proactively uncover misconfigurations and weaknesses.
Extensible modules
Add your own scanners, commands and workflows using a simple plugin interface and config files.
Privacy‑first
Runs locally without sending your scan data to external services, suitable for sensitive environments.

faq

Frequently asked questions.

A quick overview of how RavenCLI fits into your security workflow and what it can and cannot do.

What is RavenCLI?
RavenCLI is an open source, terminal‑based ethical hacking tool that combines recon, scanning and hardening tools into one interface.
How is it different from OpenCode?
While OpenCode focuses on coding assistance, RavenCLI applies a similar agent approach to security tooling with flows tuned for discovering misconfigurations and vulnerabilities.
Is RavenCLI legal to use?
RavenCLI is designed for ethical hacking and defensive security; only run it against systems you own or are explicitly authorized to test and always follow local laws and your organization’s policies.
What tools are included?
The toolkit focuses on recon and inspection: network and port scanning, service fingerprinting, misconfiguration checks and customizable command chains that you can adapt to your environment.
How do I contribute?
Fork the repository, create a branch, add modules or improvements and open a pull request; you can also help by filing issues and sharing reproducible bug reports.
How do I update RavenCLI?
Use the same package manager you installed it with or rerun the install script; the tool can also surface update notices if you enable them in the config.
